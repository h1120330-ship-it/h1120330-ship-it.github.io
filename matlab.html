二、影像取樣與縮放練習

實作原理：矩陣索引操作

影像在 MATLAB 中是以二維（或三維）矩陣儲存。透過調整索引的間隔（Sampling Interval），我們可以快速改變影像的大小與解析度。例如 x(1:3:end, 1:3:end) 代表每隔 3 個像素取一個點，藉此達成影像縮小的效果。

實作程式碼：

[x, map] = imread('mandrill.tif');
% 不同的取樣間隔生成版本
x3 = x(1:3:end, 1:3:end); % 間隔 3
x5 = x(1:5:end, 1:5:end); % 間隔 5
x7 = x(1:7:end, 1:7:end); % 間隔 7

% 顯示結果
subplot(1,2,1); image(x); title('取樣間隔:1');
subplot(1,2,2); image(x3); title('取樣間隔:3');
