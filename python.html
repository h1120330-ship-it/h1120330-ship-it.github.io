Python 自動化控制：
影音處理與語音辨識整合

從基礎語法到高階 API 應用，建構一個能自動下載、裁切並將 YouTube 影音內容轉化為雲端歸檔的自動化系統。

壹

課程核心學習重點

基礎語法與判斷

深入掌握 if 條件判斷、比較運算子與邏輯運算子 (and, or, not)，建立嚴密的程式判斷邏輯。

資料結構進階應用

學習 Python 核心資料結構，包括字典 (Dictionary)、集合 (Set) 與數組 (Tuple) 的操作與差異。

語音與文字處理

運用 SpeechRecognition 等套件進行聲音頻率捕捉與文字轉譯，處理複雜的非結構化數據。

雲端資源整合

實作 Google Colab 與 Google Drive 的對接，學習如何在遠端環境下進行自動化檔案存取。

貳

期末專題：YouTube 自動化處理系統

整合多項 Python 套件，實現一鍵式的影音處理工作流。

專題構想

本專案旨在簡化 YouTube 影音處理流程。使用者僅需提供網址與確切的時間段，系統即會自動完成下載、精確裁切，並可選擇轉為影片或純音訊格式，最後自動分類存放至雲端硬碟。

使用技術棧
pytubefix (下載核心) moviepy (影像處理) SpeechRecognition (語音辨識) Google Drive API
參

核心功能實作說明

Step 01.

下載與解析

讀取 YouTube 網址，利用 pytubefix 獲取最高畫質串流，並暫存於環境中。

Step 02.

影像裁切與調整

根據使用者輸入的時間點進行剪輯，同步顯示裁減後的長度並調整畫面大小與效果。

Step 03.

雲端歸檔

依照需求匯出內容，並將檔案自動存檔至雲端硬碟的對應資料夾。

心得感想與學習反思

從單純的邏輯編寫到解決實際問題的開發者思維。

在進階程式設計課程中，我不再只是學習「如何寫程式」，而是學習「如何用程式解決問題」。這次期末專題的開發讓我深刻體會到開源套件的強大，以及不同技術模組之間整合的難度。

最大的挑戰在於處理各種異常狀況，例如不正確的網址輸入或 API 限制。透過不斷的 Debug 與測試，我學會了如何優化程式的穩定性，這對我未來在軟體開發的道路上非常有幫助。
