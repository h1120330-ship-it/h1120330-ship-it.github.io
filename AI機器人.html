AI 機器人：從零建構
智慧視覺與自動化控制系統

融合 Micro:bit 嵌入式系統、Python 程式設計與 HuskyLens 視覺辨識技術，實現自動化小車與人臉識別告警應用。

壹

課程概述與研究動機

在人工智慧與自動化技術蓬勃發展的時代，理解機器人如何「感知」與「行動」至關重要。本課程透過 Micro:bit 作為核心控制器，學習如何操作各類感測器。從基礎的積木程式邏輯開始，逐步進階至 Python 腳本撰寫。

核心學習目標包含：掌握 Micro:bit 硬體架構、學習小車馬達控制邏輯、深入應用 HuskyLens 視覺模組進行人臉識別與物體追蹤，並透過無線電技術實現遠端操控。

程式力：積木與 Python 雙語轉換
視覺力：AI 辨識模組實務應用
貳

核心工具與硬體架構

使用教育界最熱門的 Micro:bit 與高效能 AI 視覺感測器。

Micro:bit V2 系統

5x5 LED 陣列 加速度計/羅盤 觸摸感測器 藍芽與無線電 蜂鳴器

Micro:bit 內建多種感測器，透過 GPIO 擴充能力，我們連接了馬達驅動板與超音波感測器，構建出具備智慧避障功能的小車模型。

參

開發實踐與程式邏輯

積木邏輯練習

透過 MakeCode 視覺化積木，快速建立「重複無限次」與「條件判斷」的概念。在小車控制中，我們實作了基本的循線與避障功能。

Python 進階控制

將積木轉換為 Python 程式碼，實作無線電 (Radio) 通訊協議，學習如何透過數據封包控制另一台 Micro:bit 的 LED 顯示。

肆

AI 視覺應用：HuskyLens

人臉辨識警報系統

利用 HuskyLens 的人臉辨識演算法。當偵測到已學習的 ID 時，系統會透過 I2C 通訊通知 Micro:bit，播放成功音效並顯示綠色指示圖示；反之則觸發警報。

功能整合清單

物體追蹤功能

自動追蹤指定顏色或形狀之物體，連動馬達驅動。

無線電遙控 (Radio)

藉由封包傳送控制訊號，實現高效率的無線控制。

超音波避障

精準計算物體距離，實現掃地機器人邏輯避障。

學習總結與展望

從單純的硬體邏輯到 AI 視覺識別，見證技術整合的力量。

這堂「AI 機器人」選修課帶領我進入了嵌入式系統與電腦視覺的世界。從最初接觸積木程式的驚奇，到後來能熟練地使用 Python 進行數據處理，我深刻體會到邏輯思考與實作調試的重要性。

特別是在實作 HuskyLens 時，我學會了如何優化感測器的閾值，讓辨識在不同光源下都能穩定運作。這份經驗對我日後開發各類自動化系統有著極大的啟發與幫助。
